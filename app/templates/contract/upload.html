Comment créer le modèle

  1. Exporter le contrat au format .odt
  2. Ouvrir le document au format .odt dans LibreOffice
  3. Placer les balises de la page précédente à l'endroit souhaité dans le fichier (#object_number, #show_name...)
  4. Enregistrer le document au format .odt
  5. Envoyer le fichier.


Si certaines balises ne sont pas remplacées

  • Assurez vous que le fichier est bien au format .odt
  • Utiliser LibreOffice pour modifier et enregistrer le fichier (Word a tendance à rendre les fichiers .odt illisibles)
  • Réecrire toute la ligne sur laquelle est la balise
